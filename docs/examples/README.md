# Examples

Examples illustrate the intended resource model and review experience. They are not yet guaranteed to match executable schemas, and component names in the sample plan are placeholders rather than supported products.

- [Platform request](platform-request.yaml)
- [Hardware inventory](inventory.yaml)

- [Planning audit event](audit-event.yaml)

The example represents a local air-gapped coding/chat environment for a small team. Its purpose is to show separation of intent, facts and decisions—not to recommend a real stack.
